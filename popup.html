Note: Each set of keyword options is on a new line with the format for each being:
keyword, color, [flash], [timer], [border width], [opacity], [all frames]
Where keyword and color are required. If you type in 'flash' as the third word, then the box will blink at interval set by the optional parameter 'timer'. You can also specify custom border width, opacity and 'all frames (true/false)' values that will override the default values above.
